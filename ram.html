What is RAM?

RAM stands for Random Access Memory. RAM is a critical component for computer functionality. It holds and loads programs and instructions in from permanent storage for a temporary amount of time. RAM is considered volatile memory as its contents are lost upon a power-off situation. ROM or Read-Only Memory is used for things like the BIOS and is considered non-volatile. Generally, having larger capacity RAM installed leads to better overall PC performance.
