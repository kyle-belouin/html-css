CENTRAL PROCESSING UNITS (CPU)

Overview and Functions

Overview

  • The CPU is the electronic circuitry within a computer that caries out instructions of computer programs.
  • This is done using basic arithmetic, logic, control and input output (I/O) operations originating from the instructions in a program.
  • The term "CPU" has been used since the early 1960s.
  • CPUs have gotten smaller and more efficient and continue to do so as time goes on.
  • Moore's Law notes that a limit of CPU speed does exist as the number of transistors in a CPU doubles every two years
